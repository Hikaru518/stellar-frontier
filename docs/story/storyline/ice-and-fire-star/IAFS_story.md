本篇优先强调紧急氛围与决策压迫感；聚落风貌、历史、建筑、遗迹与地标细节拆分到独立 wiki 维护。

## 第一阶段：坠落与立足

### 开场局势

先是一道失真的警报音，接着整艘飞船像被看不见的手拧断。主舰体拖着长长火尾掠过云层，在冰与火交界的地表擦出一条发亮伤痕，最终失控滑停在当地人口中的 ['灰烬霜带']。

镜头越过破裂舷窗：左侧是吞没地平线的白噪风暴，右侧是把岩层烧到通红的火山雨。世界按残酷而精准的节拍切换 ['霜爆-热浪']，前一分钟结霜，后一分钟灼热。对本地居民来说，这是会直接致命的气候窗口；对穿宇航服的探险队来说，温度本身可被恒温系统抵消，但每次跨区行动都会持续吞噬宇航服能源。舰内 [发动机核心] 失压，[备用电源] 跳到红线，生命支持进入倒计时。

[通讯阵列] 只有断续噪点，[地图] 只剩 ['过期轮廓']，没有 [可靠坐标]，没有友军回波，也没有重来一次的机会。队伍此刻面对的不是“下一站去哪”，而是“第一小时谁能活下来”。

而真正让人发冷的，是随后拼出的第一条结论：这片大陆原本并不属于极寒或极炎，它是被某种长期“分流”过程，硬生生撕成了两极。

### 情境

- 坠毁后的前 10 分钟，玩家先在 [通讯阵列] 里点名确认 5 名队员状态：有人轻伤、有人装备丢失、有人携带的传感器还能工作但电量告急。第一步不是探索，而是决定“谁留守修船、谁外出回收”。
- 前 30 分钟的外勤会连续遇到三类即时风险：
  - 低温窗口：本地人外出死亡率陡增，救援与交易活动急剧收缩；
  - 高温窗口：地表热流上冲，设备与载具故障率提升；
  - 交替瞬间：地表出现热胀冷缩裂缝，路径会突然失效。
- 探险队员穿宇航服不直接承受低温/高温伤害，但会持续消耗恒温与过滤系统能源；拖延越久，能源压力越高。
- 玩家可与环境进行基础交互：拆解坠毁舱板制作临时风障、在热岩旁布置散热片反向导流、用应急信标标记“可往返安全点”。这些交互直接决定队员能否活着跑完第一轮侦察。
- 第一批可用线索来自“看得见的异样”：
  - 冰侧反光点并非自然冰晶，而是人工打磨过的 [风障塔列] 反射片；
  - 火侧等距火点并非野火，而是矿道通风口的维护火；
  - 交界地面出现双重蹄印，提示曾有熟悉窗口节律的本地迁徙路径。
- 玩家通过“回收日志 + 地表痕迹 + 短时信号”拼出两条可行引导：
  - 冷端引导：追踪反射片与风向旗残标，可接入 [霜湾聚落] 外围巡线道；
  - 热端引导：沿矿渣辙痕与通风火点，可接入 [烬炉城寨] 外围矿道口。
- 到达前的最后一段路仍建议卡 ['霜爆-热浪'] 回落窗推进：玩家需要在通话中下达“停-走-掩蔽”节奏指令，以降低宇航服能耗与意外损耗，并尽量与本地护送窗口对齐。

### 第一阶段行动指引（强提示）

- 优先级 1：稳住生命支持，先保 [备用电源]。
- 优先级 2：回收双环境防护材料，未完成前不进入深区。
- 优先级 3：建立首条补给线，先用环境线索锁定通往 [霜湾聚落] 或 [烬炉城寨] 的一条可重复路线。
- 优先级 4：所有跨界行动必须卡温差窗口执行。
- 失败代价提示：盲目推进会直接触发伤害、掉队或补给崩盘。

### 冲突

- 队伍必须尽快接触当地聚落拿到区域生存装备，否则无法进入冰区或火区深处。
- 但当前情报互相矛盾：
  - 冰侧出现不自然反光与耐寒兽拖痕；
  - 火侧出现等距火点与矿渣辙痕；
  - [地图] 仅有 ['过期轮廓']，无法确认哪条路更安全。
- 且两条路都带代价：冷端路径更稳但绕行更远，热端路径更快但会连续穿过高热危险段。玩家必须在“低风险慢推进”和“高风险快接触”之间立刻做选择。

### 选择

- 玩家做第一轮派遣决策：
  - 全员去 [霜湾聚落]（稳健集中）
  - 全员去 [烬炉城寨]（激进集中）
  - 分两队并行（信息最大化）
- 无论哪种路线，都必须抓住短暂温差回落窗口穿越交界带；错过窗口会触发额外消耗或状态惩罚。

### 后果

- 全员去 [霜湾聚落]：生存稳定性更高，但火区情报明显滞后，主线推进偏慢。
- 全员去 [烬炉城寨]：高能材料获取更快，但事故与损耗风险显著上升。
- 分队并行：冰火两侧信息同步增长，利于后续拼图，但通讯调度复杂且容错最低。
- 抵达任一聚落后，当地不会立刻信任外来者；玩家需用资源、工具或劳务换取落脚资格，才可解锁基础装备与地方传闻。

### 阶段收束（第一阶段产出）

- 至少一条稳定补给线（[霜湾聚落] 或 [烬炉城寨]）。
- 一套可持续的双环境基础装备方案。
- 第一段不完整坐标线索（需后续阶段校准）。
- 对 [麦克斯韦妖精] 的初步认知：强大、难以正面对抗、可能在守卫 [门域]。
- 一条新的可疑假说：['霜爆-热浪'] 可能并非纯自然天气，而是 [门域] 分流过程在地表的脉冲外显。

### 阶段目标（复盘）

- 在冰火交界的极端环境中建立最低生存能力。
- 与至少一个聚落建立联系，获得区域生存装备与本地情报。
- 拿到第一批“可疑但不完整”的坐标信息。

## 第二阶段：双区调查与生态战斗

### 开场局势

队伍在第一阶段站稳脚跟后，立即被推入更高压调查期。风暴变强、补给变紧、误判代价放大。冰区与火区回传的证据彼此冲突，迫使玩家在不完整情报中继续押注；怪物战斗高频化，生物分化与环境极端化互相放大，拖延本身就是风险。

调查逐步显示：生态不是“自然渐变”，而更像被长期冷热分流重塑。冰原种群倾向低温机动与保温结构，火山种群倾向抗热爆发与地形破坏，交界稀有种则出现明显的节律拟态特征。

### 情境

- 调查队需要在冰区与火区同步采集线索，拼出可用的 [坐标碎片]。
- 怪物战斗成为主线推进的一部分：关键样本与装备材料只能通过狩猎或防卫战获得。
- 两地聚落开始发布狩猎委托，承诺用工艺、情报和通行权限交换战斗成果。
- 玩家可正式进入聚落核心区开展“人和地点”的调查，不再只是在外围接单：
  - [霜湾聚落] 可接触人物：
    - [风障记录员]：提供 ['霜爆-热浪'] 历史窗口表与风向异常注记；
    - [冰壳工匠]：要求玩家交付 [冰壳板] 才共享冷端防具改造图；
    - [边巡队长]：只在完成一次交界护送后，透露旧巡线道坐标。
  - [烬炉城寨] 可接触人物：
    - [熔井监工]：给出热流波峰时间，交换条件是先处理矿道突发；
    - [赤砧锻师]：以 [熔鳞] 和 [灼腺] 为交换，开放耐热装备升级；
    - [矿道领航员]：提供“塌方前兆图”，但会隐瞒一条高风险捷径。
- 聚落内剧情交代重点：
  - 两地都承认“同源祖地”存在，但对 [温带母村] 史实解释互相否定；
  - 村民口述与遗迹记录都指向“门域节律先异常，气候才失控”；
  - 关于妖精的共识是“它们在守门”，分歧是“守的是秩序还是牢笼”。
- 聚落周边可探索区域与情报：
  - [风障塔列]（霜湾外缘）：可拿到风向反演数据，定位冷端安全窗口；
  - [静潮仓]（霜湾后勤区）：可查到历史断供记录，推测灾变加速节点；
  - [熔井回廊]（烬炉中层）：可获取热流脉冲日志，验证火端异常规律；
  - [灰桥驿台]（烬炉外联点）：可截获跨区货单，发现交界运输被人为限流；
  - [灰烬霜带] 旧路：可追踪 [灰烬霜鹿] 蹄印，获得一条低暴露穿越线。

### 冲突

- 资源与时间不足以支持“无差别探索”，玩家必须决定优先调查方向与狩猎目标。
- 若缺少对应区域材料，队伍会在冰区/火区持续吃环境惩罚。
- 若战斗投入过高，主线调查节奏会被拖慢；若战斗投入过低，装备与样本不足又会卡住主线。
- 两地关键人物给出的妖精情报互相矛盾：一方称其在“维持温差秩序”，另一方称其在“制造温差统治”。
- 周边地点的记录也存在断层：日志被删改、坐标被重写、货单时间戳对不上，玩家必须判断哪些线索可信。

### 选择

- 区域推进策略：
  - 偏冰区深挖：先保障耐寒体系与冷端样本。
  - 偏火区深挖：先保障耐热体系与高能样本。
  - 双区同步校验：并行采样，提升信息完整度但增加调度压力。

- 聚落探索策略：
  - 先做霜湾线：优先锁定风障窗口与冷端稳定补给，再推进火端。
  - 先做烬炉线：优先拿热流日志与高能工艺，再反推冷端异常。
  - 双聚落交叉取证：同步跑“人物口述 + 地点日志 + 实地痕迹”，最快拼出高可信情报链，但最吃时间。

- 战斗与委托策略：
  - 冰区狩猎线：
    - [霜脊狼] -> 掉落 [霜狼皮] / [寒腺结晶]（耐寒内衬与低温药剂辅材）
    - [裂冰甲虫] -> 掉落 [冰壳板] / [裂冰前肢]（低温护甲与破障工具材料）
    - [白噪鸦群] -> 掉落 [回声翎] / [空腔喙骨]（信号校准与拾音器改造件）
    - [冻湖潜鲼] -> 掉落 [低温脂膜] / [冰鳍纤维]（保温涂层与防滑组件）
    - [风切鹿] -> 掉落 [风纹角片] / [导流筋膜]（风障预测与轻量护具辅材）
  - 火区狩猎线：
    - [熔鳞蜥] -> 掉落 [熔鳞] / [灼腺]（耐热层与热武器改件）
    - [硫烟飞螳] -> 掉落 [硫囊] / [薄刃翅骨]（过滤组件与轻量切割件）
    - [炉喉犬] -> 掉落 [耐压喉骨] / [炽牙]（高温管路与耐热钻头辅材）
    - [赤灰钻虫] -> 掉落 [炭芯节] / [钻节壳]（稳定燃烧媒介与抗压结构件）
    - [喷炬翼蝠] -> 掉落 [火雾囊] / [耐灼翼膜]（短时热障与滤层辅材）
  - 交界稀有线：
    - [相变捕食者] -> 掉落 [相变膜] / [界面腺核]（双环境关键材料与相位实验核心）
    - [灰烬霜鹿] -> 获取 [双温蹄晶] / [灰霜毛束]（跨界导航与伪装披覆材料，非敌对追踪）
    - [门影蜉蝣] -> 掉落 [门纹粉] / [脉冲翅屑]（节律观测样本与谐振校准辅料）
    - [节拍仿行兽] -> 掉落 [仿律腺] / [步态棘节]（协议研究与反制诱导模块）
    - [阈值寄生藤] -> 掉落 [阈值孢囊] / [节律纤维]（隔离实验与节拍阻尼构件）

- 聚落支线选择：
  - 承接 [霜湾聚落] 委托（换取冰区工艺与捷径情报）
  - 承接 [烬炉城寨] 委托（换取火区通行许可与高能材料）
  - 同时承接两边（收益高，但时间窗口压力更大）

### 后果

- 偏冰区深挖：环境生存更稳，但火端参数不足，主线推进偏慢。
- 偏火区深挖：工程收益更快，但事故率与装备损耗明显上升。
- 双区同步校验：最容易拼出高质量 [坐标碎片]，但任何调度失误都可能引发连锁崩盘。
- 完成聚落委托会提升当地信任，并解锁下一阶段所需的 [门域] 外围情报入口。
- 若成功完成聚落人物线与周边地点调查，可额外解锁“妖精情报三联证据”（口述、日志、现场），使第三阶段沟通或潜入路线更稳定。
- 若调查偏科（只做战斗不做取证），第三阶段将出现“情报盲区惩罚”：门域节律误判概率上升。

### 阶段收束（第二阶段产出）

- 获取冰端与火端的关键样本。
- 解锁至少一条聚落高信任回报（工艺/通行/情报）。
- 拼出可校准的 [坐标碎片]。
- 明确第三阶段目标：进入门域外围，验证妖精守卫逻辑并补全离场坐标条件。
- 完成至少一条“聚落人物线 + 周边地点线”闭环，获得可用于门域研判的高可信情报包。

### 阶段目标（复盘）

- 通过战斗与调查同时建立“生态理解 + 装备升级 + 坐标拼图”。
- 让玩家直观看到生物多样性与环境变迁的因果关系。
- 以聚落支线回报反哺主线推进，形成调查与战斗的闭环。

## 第三阶段：门域接触与分歧

### 开场局势

第二阶段结束后，队伍已经拿到可校准的 [坐标碎片]，却仍缺少离场所需的关键参数。所有线索都指向 [门域]：那里是 [麦克斯韦妖精] 的活动核心，也是唯一可能补全离场条件的区域。

需要强调的是，[门域] 不是一个“单点目标”，而是错综复杂的多层区域：外围警戒环、节律过渡带与核心控制结点彼此嵌套。潜入失败往往不是因为战力不足，而是因为在错误层级做了正确动作。

但队伍不能再靠“单一路径”推进。两地聚落、队员与后勤状态都在逼迫玩家站队：是潜入、沟通，还是驱逐。第三阶段由三条支线组成，每条支线对应一种策略辅助。

在对门域残留记录的破译中，队伍首次接触到妖精文明的核心信念：它们信奉 ['智慧之神'] 与 ['多维宇宙之神']。传说这两位神赐予了 [门] 的科技，包括用于逆熵实验的核心门与跨时空的 [传送网络]。这让“沟通路线”不再只是外交尝试，而是一次接近神话级技术体系的风险接触。

技术组给出更激进推断：妖精并非只在“守门”，而是在持续控制 [门] 分流高低能粒子，长期搬运“热原子”与“冷原子”，把原本适宜气候的大陆拉成火山与冰原双极结构。

### 情境

- 主线目标：补全离场参数并进入门域外围。
- 推进方式：完成三条策略支线中的任意两条，解锁终局入口。
- 规则：不同支线组合将改变终局默认策略倾向与风险结构。
- 地图层提醒：门域行动默认跨越多个子区域（外围警戒环 -> 节律过渡带 -> 核心控制结点），每次跨层都可能触发新的巡逻或节律判定。
- 玩家实际推进路径分为四步：
  - 第一步（证据归档）：把第二阶段拿到的 [坐标碎片]、聚落日志与怪物样本统一校验，先排除一批假线索。
  - 第二步（路线预判）：基于 [风障塔列] 与 [熔井回廊] 的时序数据，生成一次可执行的门域接近窗口。
  - 第三步（队伍编组）：按当前伤势、装备与专长分组，决定谁负责侦察、谁负责交涉、谁负责掩护。
  - 第四步（首次接触）：在门域外围触发第一次“被观测”事件，系统要求玩家立即确定主策略倾向（潜入/沟通/驱逐）。
- 首次接触后，三条支线不再是纯并行任务，而是“主路线 + 备选路线”的资源博弈：你每推进一条，另外两条的成本都会上升。

### 冲突

- 时间窗口有限，无法三线完美推进。
- 资源不足以同时支持“高强度战斗 + 复杂研究 + 多线侦察”。
- 每条支线都会放大某种能力，也会放弃另一种冗余安全。
- 玩家必须做出连续抉择，而不是一次性选边：
  - 是否先用一次低风险窗口做“试探接触”，还是直接投入主行动抢进度；
  - 是否把关键材料优先投入 [共振发声器]（沟通线），还是投入边境火力与防护（驱逐/潜入线）；
  - 当聚落给出相互矛盾的情报时，是优先相信“神谕口述”，还是优先采用“技术测绘”。
- 这些抉择会提前塑形终局：你在第三阶段省下的代价，会在第四阶段以风险、时间或关系债的形式补回来。

### 选择

1. 支线 A：盲区测绘（辅助潜入策略）
   - 目标：定位 [门域] 外围警戒环与巡逻回廊的短时盲区，建立潜入窗口。
   - 关键行动：拼接 [霜湾聚落] 风向记录、[烬炉城寨] 矿道旧图与外围巡逻时间表。
   - 成功收益：解锁“低警戒潜入”选项，潜入失败惩罚降低。
   - 失败后果：触发追击事件，后续潜入警戒值上升。

2. 支线 B：七拍语法（辅助沟通策略，复杂线）
   - 目标：建立与 [麦克斯韦妖精] 的最低可用交流协议。
   - 四步流程：
     - 采样：在三处点位记录 ['七拍回声']，并完成昼夜双时段采样。
     - 对照：结合遗迹碑文与聚落口述，完成“节律-行为”映射并剔除误译。
     - 构建：狩猎稀有目标获取 [仿律腺]、[脉冲翅屑] 与 [界面腺核]，制作 [共振发声器]。
     - 实测：在门域节律过渡带执行三轮“信号-回应”测试（让路/停火/交换），并完成一次 ['神名誓约'] 语句校验。
   - 成功收益：解锁“协议通行”选项，门域敌意显著下降，并获得关于 ['智慧之神'] / ['多维宇宙之神'] 与 [传送网络] 的片段真相。
   - 失败后果：进入“误译状态”，短期关闭沟通选项并提升警戒。

3. 支线 C：边境清剿（辅助驱逐策略）
   - 目标：削弱妖精外围控制圈并压制核心控制结点外侧防线，争取强行操作窗口。
   - 关键行动：接受两大聚落联合作战委托，清除外围节点并守住回收点。
   - 成功收益：解锁“强制介入门域”选项，短时夺取控制权。
   - 失败后果：队伍状态恶化、聚落信任下降、生态波动升级。

### 后果

- 完成任意两条支线后进入终局任务。
- 组合结果会影响终局策略倾向：
  - A + B：潜入/沟通优势，整体风险更可控。
  - A + C：潜入/驱逐混合，执行力强但波动高。
  - B + C：可谈可打，灵活但资源压力最大。
- 若仅完成一条支线，主线可继续但会带着明显短板进入高风险状态。
- 走沟通优势（A+B 或 B+C）时，可触发“神话-技术双解释”分支：
  - 村民视角：妖精是在执行神谕。
  - 探险队视角：神谕可能是对高维技术规则的人格化叙述。

### 阶段收束（第三阶段产出）

- 至少两条策略支线的可用成果。
- 一套明确的门域介入路径（潜入 / 沟通 / 驱逐或混合）。
- 离场参数补全过程所需的前置条件清单。

### 阶段目标（复盘）

- 让策略选择从“偏好”变成“有代价的结构化取舍”。
- 用支线把三种策略具象化，并为终局路径提供可解释前置。

## 第四阶段：离场窗口与终局解释

### 开场局势

第三阶段结束后，队伍终于具备介入 [门域] 的条件，但窗口极短。['霜爆-热浪'] 周期进一步压缩，门域稳定时间开始跳变，飞船 [发动机核心] 只能承受一次高负荷启动。玩家必须在一次执行中完成三件事：补全离场参数、完成引擎修复、锁定 [下一星球坐标]。

与此同时，关于妖精与 [门] 的解释出现分裂：
- 神谕解释：妖精在执行 ['智慧之神'] / ['多维宇宙之神'] 的意志。
- 技术解释：所谓神谕是高维技术规则的人格化叙述；[门域] 可能在执行“粒子分流 + 逆卡诺循环”，并表现出近似第二类永动机的异常特征。

### 情境

- 终局不再提供新线索，而是结算前三阶段的全部选择。
- 玩家需决定采用哪条终局执行策略（潜入 / 沟通 / 驱逐）。
- 每条策略都能让飞船离场，但带走的“真相深度”和“代价结构”不同。
- 终局执行被拆成连续行动链，玩家需要逐步下令：
  - 行动 1（离场准备）：分配队员到“引擎修复组 / 门域行动组 / 后撤保障组”，并决定是否牺牲一部分防护库存换启动效率。
  - 行动 2（窗口判断）：在 2-3 个短窗口中选择一次主推进时机；早进窗口更稳但信息不足，晚进窗口信息更全但容错更低。
  - 行动 3（门域介入）：根据当前策略执行潜入、谈判或清剿，同时实时处理突发事件（警戒抬升、设备失灵、队员受伤）。
  - 行动 4（参数锁定）：在撤离倒计时前完成 [下一星球坐标]、[逆卡诺校准式] 与引擎点火参数的最终确认。
- 终局中至少会出现一次“二选一保全”抉择：保队员、保数据、或保窗口，三者无法同时满额保留。

### 冲突

- 时间只够执行一套主方案，无法兼顾所有价值。
- 玩家必须在“回家成功率 / 队伍完整度 / 星球后续稳定”之间排序。
- 任何方案都不是零代价，区别只在代价落到谁身上、留到何时爆发。
- 关键抉择会被强制触发：
  - 资源抉择：最后一批 [相变膜] 与 [节律纤维] 是用于提升飞船点火稳定，还是用于门域行动生还率。
  - 关系抉择：是否调用两地聚落援军；调用可提升执行力，但会把后续冲突代价转嫁给聚落。
  - 认知抉择：面对神谕解释与技术解释冲突时，是否公开“第二类永动机异常”结论；公开会动摇秩序，不公开会保留隐患。

### 选择

1. 潜入主导终局（信息优先）
   - 执行：利用盲区窗口潜入门域外围，提取 [逆卡诺校准式]、[门芯相位钥] 与关键时序参数。
   - 特点：获取真相速度快，但只能拿到“可复现方案”，无法直接带走门域核心。
   - 代价：暴露后警戒飙升，离场窗口可能被强行压缩。

2. 沟通主导终局（协议优先）
   - 执行：通过“七拍语法 + ['神名誓约']”与妖精达成临时通行协议，获得授权参数与稳定供能窗口，并请求降低冷/热粒子分流强度。
   - 特点：最稳路径，可降低门域敌意并减少生态冲击。
   - 代价：流程最长，对前置支线完整度要求最高。

3. 驱逐主导终局（控制权优先）
   - 执行：联合两地聚落发动强制清剿，夺取门域短时控制权并强行介入修复流程，同时尝试物理封停分流节点。
   - 特点：短期效率高，可直接拿到足够修复资源。
   - 代价：队伍损耗和生态波动显著上升，且真相层最薄。

### 后果

- 终局会同时给出“离场结果”与“解释立场”。

- 结局 A：速返（潜入倾向）
  - 离场：成功，航程最短。
  - 解释：偏技术解释，神谕被视为编码叙事。
  - 代价：门域关系紧张，星球后续不稳定。

- 结局 B：协返（沟通倾向）
  - 离场：成功，航程中等。
  - 解释：神谕与技术并存，保持“可解释但不被解释完”。
  - 代价：时间成本高，但队伍与星球状态最稳。

- 结局 C：血返（驱逐倾向）
  - 离场：勉强成功。
  - 解释：真相未解，叙事停在“胜利但空白”。
  - 代价：高伤亡/高创伤，后续章节负担最大。

- 结局 D：滞留修正（终局失败）
  - 离场：失败，但飞船未彻底毁坏。
  - 解释：无定论，进入补救篇章。
  - 代价：资源重置压力与窗口再争夺。

### 阶段收束（第四阶段产出）

- [下一星球坐标] 与对应置信标签。
- 当前章节解释立场标签（神谕倾向 / 技术倾向 / 混合倾向）。
- 星球遗留状态标签（友好 / 紧张 / 破碎），用于后续章节继承。

### 阶段目标（复盘）

- 把“回家”从单一胜负改为“可达成但带后果”的终局结构。
- 通过神谕解释与技术解释并存，保留世界观神秘感与讨论空间。

## 支线库

### 支线 01：失温婚约（霜湾）

- 语气基调：浪漫（在极端环境中讨论承诺与共同体）
- 情境：任务从匿名誓物线索切入。玩家会发现这不是普通护送，而是 [岚脊] 与 [曜砂] 的跨聚落婚约请求；两人希望在交界带完成会合并公开关系，以此证明“同源两地仍可共居”。
- 冲突：两地保守派都把婚约视作对既有叙事秩序的挑战。封锁令名义上是安全管制，实际上在阻断会合；玩家既要处理窗口风险，也要处理政治风险。
- 任务流程：
  - 节点 1（接单）：解析匿名中继留言，确认誓物来源与联络点。
  - 节点 2（取证）：在对侧聚落调阅巡逻记录与封锁令，判定是否存在“人为加码”。
  - 节点 3（破局）：在窗口内选择会合推进方式（高公开度 or 高隐蔽度）。
  - 节点 4（裁决）：在对峙现场决定婚约处理方式与见证安排。
- 关键抉择：
  - 公开见证：风险最高，但最有机会推动结构性和解。
  - 秘密送达：短期更稳，但无法改变两地公共叙事。
  - 暂缓婚约：即时冲突最低，但关系线受损并留下补偿成本。
- 主要耦合：影响两地关系与第四阶段“友好 / 紧张 / 破碎”标签权重。
- story-event 文件：`IAFS_side_01_frozen-vow.md`

### 支线 02：火井税单（烬炉）

- 语气基调：哲思（制度正义与生存效率的冲突）
- 情境：[烬炉城寨] 下发“火井税单”：使用主锻造炉前，必须先缴纳高危区材料与清剿凭证。其名义是安全秩序，实际是把灾变成本制度化。
- 冲突：主线同样急需这些材料推进门域前置。缴足可换制度支持但会拖慢主线，不缴或少缴则能提速但会被视作外来掠夺。
- 任务流程：
  - 节点 1（查税）：调阅税档、事故簿与配额流向，确认账面是否自洽。
  - 节点 2（实勘）：进入矿道处理高温与塌方风险，验证税额上涨是否真实用于安全维护。
  - 节点 3（问责）：在监工、锻师与矿工家属之间进行公开对质或私下谈判。
  - 节点 4（裁决）：决定税单执行方式（完整缴税 / 保留主线份额 / 公开质询）。
- 关键抉择：
  - 完整缴税：秩序收益高，主线资源压力上升。
  - 保留份额：主线推进快，热端关系与交易成本恶化。
  - 公开质询：短期收益低，但可能重塑治理规则。
- 主要耦合：影响热端信任、交易压力、第三阶段资源与通行成本。
- story-event 文件：`IAFS_side_02_firewell-tax-bill.md`

### 支线 03：灰烬拾荒者（交界带）

- 语气基调：幽默（灾难场景中的人间闹剧）
- 情境：交界拾荒队不是正规组织，而是由被边缘化者临时拼成的无政府队伍。他们造出“冰火鸳鸯锅车”后卡在窗口外，锅能工作、人却出不去。探险队抵达拾荒营地后，支线正式触发并进入“找食材”委托。
- 主要人物：
  - 队长 [寒栓]：来自 [霜湾聚落]，却是急躁火爆的性格；因长期不符本地期待而离开故地，执着冷端食材。
  - 副队 [炽砧]：来自 [烬炉城寨]，却极度冷静克制；同样因性格与环境期待不匹配而离开故地，坚持热端食材优先。
  - 成员 [簿灰]：计量狂，负责记账和配比。
  - 成员 [回针]：路痴猎手，找路能力波动极大。
  - 成员 [铆铃]：锅具技师，只在设备参数上有耐心。
  - 成员 [白噪]：观察员兼段子手，负责把灾难写成笑话。
- 团队目标：在 [灰烬霜带] 建立一个松散乌托邦，收纳同样被主流秩序排斥的离经叛道者。
- 冲突：本地人受 ['霜爆-热浪'] 直接压制，拾荒队无法长距离外出；探险队能靠宇航服恒温跨区，但每次采集都会消耗宝贵能源。
- 任务流程：
  - 任务 1（找食材）：玩家在冷端与热端食材线之间取舍，决定先满足队长还是副队，或冒险双线全做。
  - 任务 2（做菜）：玩家协助处理食材、备料和烹煮，并在“地球家乡菜”方案中三选一（川味麻辣锅 / 法式红酒炖牛肉锅 / 墨西哥辣可可炖锅）。
  - 任务 3（火锅派对）：派对中选择进食策略；若冷热混吃，触发腹泻事故并留下舆情笑柄。
- 后果：
  - 双线完成且安全收束：冷端与热端奖励全拿，交界协作关系提升。
  - 仅完成单线：获得对应负责人奖励（[寒栓] 冷端包或 [炽砧] 热端包），另一侧关系不增长。
  - 冷热混吃：保留基础奖励，但出现短期状态惩罚并写入“饮食事故”历史键。
 - story-event 文件：`IAFS_side_03_ashland-scavengers.md`

### 支线 04：孢子禁区（门域外围）

- 语气基调：惊悚（恐惧来自“看不见的感染”和“谁先失常”）
- 情境：两地聚落同时发来紧急告警：门域外围出现一片会“呼吸”的 [孢子禁区]。前一天还只是薄雾，第二天已吞没整条补给路，接触者会出现短时幻听与节律错觉。
- 冲突：若不尽快封锁，孢子将沿物流路径扩散到居住区；但封锁行动会占用潜入准备资源，直接挤压第三阶段窗口。
- 任务流程：
  - 节点 1（侦测）：在禁区边缘布设三组过滤桩，记录孢子浓度曲线与 ['七拍回声'] 的同步性。
  - 节点 2（症状排查）：护送一名疑似感染工人回到临时隔离点，并在“救治优先/情报优先”之间做选择。
  - 节点 3（封锁执行）：可选择火墙封边、低温压制或混合方案；不同方案对生态与后续通行影响不同。
  - 节点 4（夜间守线）：防止 [阈值寄生藤] 在封锁带重新开囊，熬过一整轮窗口才算封锁成立。
- 选择：
  - 优先民生封锁：投入主力完成全域封边，确保聚落安全。
  - 优先门域行动：只做最小封锁带，把资源留给主线推进。
  - 分段封锁：先保居住区通道，再逐段回收禁区边缘。
- 后果：
  - 优先民生封锁：聚落长期支持显著上升，居民线信任解锁；代价是主线推进至少延后一轮窗口。
  - 优先门域行动：主线节奏最佳，但后续会周期性触发“孢子回潮”惩罚事件。
  - 分段封锁：风险与收益居中；若中段失守，会出现“已净化区域复染”的心理打击事件。

### 支线 05：妖精弃巢（生态观察）

- 语气基调：科幻（未知感来自规则、结构与可复现实验）
- 情境：侦察队发现一处被遗弃的 [妖精节点]。节点内部没有尸骸和战斗痕迹，只有规律亮灭的几何腔室，像一台被暂停的高维实验装置。
- 冲突：节点可能是陷阱，也可能是“主动留给观察者的样本”。处理方式将决定你拿到的是可复现知识，还是一次性战利品。
- 任务流程：
  - 节点 1（拓扑测绘）：绘制节点三层结构，确认外壳、过渡舱、核心井的能量流向。
  - 节点 2（非接触采样）：利用 [门纹粉] 与 [脉冲翅屑] 做谐振扫描，建立第一版节律模型。
  - 节点 3（受控实验）：选择一个窗口做“小功率激活”，验证节点是否与 [门域] 主体同频。
  - 节点 4（封存或拆解）：决定留下监测装置长期观测，还是一次性拆走核心部件。
- 选择：
  - 潜行采样：低扰动获取高质量数据，优先科研价值。
  - 快速拆解：高风险高回报，优先材料与即战力。
  - 标记回收后延迟处理：先挂监测标记，等待更完整窗口再进入。
- 后果：
  - 潜行采样：获得稳定的 [门芯相位钥] 线索与“节律-能量”对照表，利于第三阶段沟通与潜入。
  - 快速拆解：可拿到稀有核心材料并立刻强化装备；代价是门域警戒值显著上升。
  - 延迟处理：短期收益最低，但可在后续章节触发“连续观测数据”加成，提升终局参数置信度。

### 支线 06：双村停火（社会线）

- 语气基调：严肃（核心是责任、记忆与政治代价）
- 情境：[霜湾聚落] 与 [烬炉城寨] 因窗口物资分配与历史伤亡叙事公开对峙。双方都拿出“牺牲名单”要求探险队表态：你支持谁，谁就开放核心通道。
- 冲突：站队可换即时高回报，却会把另一侧推向长期敌对；强行中立则可能两边都失去信任。
- 任务流程：
  - 节点 1（听证）：分别参加两地公开陈述，收集各自的损失账本与口述证词。
  - 节点 2（核证）：前往交界旧仓核对过往配给记录，确认“谁先违约”是否属实。
  - 节点 3（提出方案）：在停火桌上提交一份三条款草案（窗口轮换、伤员互送、信息互验）。
  - 节点 4（执行担保）：选择由谁担保首轮停火执行，探险队自己或单侧聚落。
- 选择：
  - 促成停火：推动双边签署临时协议并承担担保责任。
  - 偏向 [霜湾聚落]：换取冷端全线支持。
  - 偏向 [烬炉城寨]：换取热端全线支持。
- 后果：
  - 停火成功：解锁 [双村联合补给网] 与跨区低损通道；但后续任何失误都会优先归咎于你。
  - 偏向霜湾：短期生存稳定显著提升，火端关系急剧恶化，第三阶段强攻成本上升。
  - 偏向烬炉：短期战斗与工程效率暴增，冷端协作断裂，长期舆情与救援支持下降。

### 支线 07：最后一班矿车（战斗撤离）

- 语气基调：哲学（在“救多数”与“救眼前”之间追问价值排序）
- 情境：火区矿道连续塌陷，最后一班矿车被困在高温走廊。车上既有矿工，也有用于维持聚落供能的核心矿样；你无法在一次行动里同时完美保全两者。
- 冲突：救人优先会牺牲物资与后续供能，保矿优先则会把“可计算的效率”凌驾于“不可替代的生命”。
- 任务流程：
  - 节点 1（断面评估）：测算走廊温度、坍塌概率与回撤时间，判断可执行时长。
  - 节点 2（护送编组）：分配队员负责前锋清场、车尾掩护与医疗应急。
  - 节点 3（临界抉择）：在撤离中途触发二选一事件——救下掉队伤员或保住装载矿样的车厢。
  - 节点 4（终点裁定）：抵达 [灰桥驿台] 后，由玩家公开说明取舍逻辑，决定责任叙事归属。
- 选择：
  - 先清场后撤离：尽量降低途中伤亡，但可能错过最佳回撤窗。
  - 边打边撤：速度更快，容错更低，任何失误都会放大损失。
  - 人命优先协议：提前声明“可弃矿不可弃人”，并按此重排战术。
- 后果：
  - 成功高保全：获得高能矿样本与 [烬炉城寨] 作业支持，并额外提升社会线公信力。
  - 人命优先成功：矿样收益下降，但会在后续剧情中解锁“矿工证词线”，强化停火与民生支持。
  - 失败撤离：队伍状态恶化与 [烬炉城寨] 信任下降；且这次失败会被写入两地争端叙事，长期放大政治裂痕。
