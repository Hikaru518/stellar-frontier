# 最小用例（MVP 可实现版）

## 1. 概述

玩家作为远程指挥者，在坠毁后的未知星球上通过通讯调度队员调查、采集、修复和处理危险，最终修复折跃仓、取得折跃坐标与燃料，带队返回家园。

这份用例只保留一条可稳定通关的 `MVP` 主线，并保留少量属性捷径。语言解锁、轻量村落对象与单次关键交易属于 `MVP`；更复杂的贸易、驯兽与多层支线内容留到后续版本。

## 2. 设计意图

- 让玩家体验“远程指挥一支落难小队，从混乱到建立返航方案”的过程。
- 让探索结果不仅带来资源，还带来新的可执行决策。
- 保证存在一条不依赖随机和高属性的保底通关路线。
- 高属性角色、高武力或额外探索应当提供捷径，而不是成为唯一解。

## 3. 核心概念与术语

- **回家主线**：修复折跃仓并完成一次折跃返航的主目标。
- **保底路线**：任何开局都可完成的稳定主线路径，不依赖随机掉落或稀有属性。
- **捷径路线**：高武力、高社交或额外道具带来的降风险、降耗时方案。
- **维修技术**：通过飞船日志与技术文档获得的关键标签，用于修复熔炉与折跃仓。
- **野外急救**：通过医疗舱获得的关键标签，用于救助受伤目标与处理受伤状态。
- **可重复学习**：主线必需知识不会只绑定在单个队员身上；只要原始文档仍在，其他队员也可补学。
- **高温开采设备**：进入火山并采集黑曜石的主线工具。
- **折跃仓修复套件**：修复折跃仓所需的关键部件。
- **外星粘液燃料**：折跃仓燃料来源，由巢穴主线取得。

## 4. 核心循环与玩家体验

### 4.1 玩家旅程
1. 坠毁开场后，队员来电报告现状，主线任务“回家”建立。
2. 玩家命令队员多次调查坠毁现场，逐步获得基础物资、损坏折跃仓现状、雷达线索与维修技术。
3. 玩家通过调查雷达获得稳定的主线方向信息，并决定先去哪里。
4. 玩家派队员去关键地点调查，逐步取得修复折跃仓所需的知识、材料和燃料。
5. 玩家在通话中持续处理风险：受伤、入口守卫、资源不足、路线选择。
6. 玩家完成折跃仓修复，取得折跃坐标与燃料，触发返航结局。

### 4.2 目标体验

- 前半段是“信息不足下的探索调度”。
- 中段是“把分散线索整理为返航方案”。
- 后段是“完成几个关键前置后，明确朝终局推进”。

## 5. 主线闭环（保底路线）

### 5.1 开场与初始发现

队员在坠毁后主动联系玩家，报告：

- 有人受困但暂时可行动
- 坠毁现场可调查
- 当前唯一明确目标是找到回家方法

坠毁现场通过多次调查渐进揭示主线：

1. **第一次调查**：获得生存物资，并确认折跃仓已经损坏，当前无法直接返航。
2. **第二次调查**：恢复基础雷达，开始获得大致方位线索。
3. **第三次调查**：发现维修技术文档，并得知医疗舱从主船体脱落到了别处。

第一次调查后，稳定获得：

- 基础资源：食物、简易武器、基础工具
- 损坏的折跃仓
- 折跃仓无法直接启动的现状说明

第二次调查后，稳定获得：

- 损坏但可工作的基础雷达
- 若干大致方位线索

第三次调查后，稳定获得：

- 日志与技术文档
- 医疗舱脱离船体掉落在某处的信息

日志与技术文档会让一名调查队员获得 `维修技术` 标签。

这些日志与文档会被带回基地并保留下来，因此在 `MVP` 中它们属于可重复学习的知识源：

- 任何回到坠毁点并执行“学习资料”的队员，都可以补学 `维修技术`
- 这样即使首位掌握该标签的队员后续受伤或失联，主线也不会硬锁

### 5.2 雷达线索

基础雷达采用“主线线索稳定给、额外发现可随机”的原则。

主线必需的稳定线索包括：

- 北边存在生命迹象
- 东边存在旧金属反应
- 南边存在高温反应
- 附近存在旧医疗应答信标

这些线索分别对应：

- 北边生命迹象：首先引导玩家前往外星村落
- 东边旧金属反应：后续可对应旧飞船遗迹线索
- 南边高温反应：对应火山与黑曜石来源
- 医疗信标：对应损坏医疗舱的位置

额外发现可以随机给出，但不承担主线前置职责。

### 5.3 保底主线步骤

1. 调查坠毁现场，完成三次渐进揭示，获得 `维修技术` 与雷达线索。
2. 前往丘陵，进行持续采集，获得铁矿，并在若干轮采集后保底获得 `稀有矿石样本`。
3. 前往外星村落。若高社交成功，则直接学会外星语言；若失败，则通过肢体沟通获得 `罗塞塔` 的大致方向与地形特征。
4. 如未直接学会外星语言，则前往罗塞塔装置，解锁 `外星语言`。
5. 返回外星村落，与商人用 `稀有矿石样本` 稳定交易获得 `高温开采设备`。
6. 与村民交流，获得 `受伤村民营地` 的有效线索。
7. 前往医疗舱，获得基础药品，并通过医疗文档学习 `野外急救`。如首位学习者无法继续行动，其他队员也可回到医疗舱补学。
8. 前往火山，先调查环境，明确这里需要 `高温开采设备` 才能安全采集黑曜石，并通过残骸散落轨迹反向发现旧飞船遗迹方向。
9. 使用 `高温开采设备` 在火山一次采集成功，获得 `1 份黑曜石`。
10. 前往损坏熔炉，以 `维修技术 + 铁矿` 修复熔炉，并制造 `折跃仓修复套件`。
11. 前往旧飞船遗迹，调查失效驾驶舱，导出一段已缓存的返航坐标，获得 `折跃坐标`。
12. 前往受伤村民营地。使用 `药品` 或 `野外急救` 完成救援，获得 `诱饵`，并明确得知诱饵可引开兵蚁守卫。
13. 前往外星生物巢穴：在入口观察兵蚁巡逻与危险提示；若没有诱饵，可撤回另找办法；若持有诱饵，可通过通话明确选择使用。
14. 进入孵化室后，再次使用诱饵引开守卫，采集 `外星粘液燃料`，并直接成功撤离。
15. 返回坠毁点，完成折跃仓终局三步：修复仓体结构、注入燃料、输入坐标并确认启动，触发返航结局。

这条路线不依赖：

- 高武力
- 驯兽
- 随机掉落

## 6. 关键地点

### 6.1 坠毁现场

- 功能：开场、建立主线、取得维修技术与雷达线索入口
- 回报：基础资源、折跃仓、雷达、日志文档
- 备注：这是主线必经地点

### 6.2 丘陵

- 功能：提供铁矿与稀有矿石样本来源
- 回报：铁矿；持续采集若干轮后保底得到 `稀有矿石样本`
- 风险：开采或调查时可能触发受伤或狩猎相关事件

### 6.3 外星村落

- 功能：语言分叉、交易与营地线索入口
- 初访：谨慎但不敌对
- 无语言时：
  - 高社交成功可直接学会 `外星语言`
  - 失败也不会卡死，会通过肢体沟通得到 `罗塞塔` 大致方位
- 有语言时：解锁村内对象
  - 商人：用 `稀有矿石样本` 交易 `高温开采设备`
  - 村民：提供 `受伤村民营地` 的有效线索

### 6.4 罗塞塔装置

- 功能：保底解锁 `外星语言`
- 备注：只承担语言解锁职责，不直接承担其他主线功能

### 6.5 损坏熔炉

- 功能：主线制造地点
- 前置：维修技术
- 回报：可制造 `折跃仓修复套件`

### 6.6 火山

- 功能：黑曜石来源，并反向抛出旧飞船遗迹线索
- 前置：高温开采设备
- 调查回报：明确“需要高温开采设备才能安全采集黑曜石”，并发现残骸散落轨迹
- 采集回报：一次获得 `1 份黑曜石`

### 6.7 旧飞船遗迹

- 功能：折跃坐标来源
- 场景：半埋沙中的残骸与失效驾驶舱
- 回报：导出半损坏导航记录中的一段返航缓存坐标，并附带一小段“返航未遂”的叙事回响

### 6.8 医疗舱

- 功能：恢复与标签解锁
- 回报：药品、`野外急救`
- 备注：医疗文档同样属于可重复学习资料，其他队员后续可补学 `野外急救`

### 6.9 受伤村民营地

- 功能：验证医疗能力与提供诱饵
- 初见：明显受伤、急需救助的村民
- 前置：`药品` 或 `野外急救` 均可完成救援
- 回报：`诱饵`，并明确说明这种诱饵可引开兵蚁守卫

### 6.10 浅滩湿地

- 功能：额外容错诱饵来源
- 获取方式：调查后触发“捕获小型生物”事件
- 备注：不是主线首次诱饵来源，而是补救与额外探索奖励

### 6.11 外星生物巢穴

- 功能：折跃燃料来源
- 最小结构：入口 -> 孵化室 -> 取燃料撤离
- 入口：有兵蚁巡逻与危险提示；无诱饵时可观察后撤；有诱饵时可在通话中明确选择使用
- 孵化室：外星粘液可采集，但有守卫看守；稳定通过诱饵引开守卫后采集
- 回报：`外星粘液燃料`

## 7. 关键道具与标签

| 名称 | 来源 | 用途 | 是否主线必需 |
| --- | --- | --- | --- |
| 维修技术 | 坠毁现场日志文档 | 修复熔炉、修复折跃仓 | 是 |
| 铁矿 | 丘陵 | 修复熔炉、制造设备 | 是 |
| 稀有矿石样本 | 丘陵持续采集，若干轮后保底获得 | 与村落商人交易高温开采设备 | 是 |
| 高温开采设备 | 村落商人交易 | 火山采集黑曜石 | 是 |
| 黑曜石 | 火山 | 制造折跃仓修复套件 | 是 |
| 折跃仓修复套件 | 熔炉制造 | 修复折跃仓 | 是 |
| 折跃坐标 | 旧飞船遗迹 | 启动折跃 | 是 |
| 野外急救 | 医疗舱 | 救助营地村民、处理受伤 | 是 |
| 药品 | 医疗舱 | 恢复受伤状态 | 否，但强建议 |
| 诱饵 | 受伤村民营地；浅滩湿地为额外来源 | 引开兵蚁，推进巢穴路线 | 是（保底路线） |
| 外星粘液燃料 | 巢穴孵化室 | 作为折跃燃料 | 是 |

## 8. 捷径与可选优化

以下内容存在，但不承担保底主线职责：

- 高武力队员可在部分巢穴或危险场景中尝试强行突破，但不作为保底路线。
- 高社交队员可在第一次村落接触时直接学会 `外星语言`，跳过 `罗塞塔` 线。
- 雷达升级可减少探索试错，但不是主线必要前置。
- 额外支线地点可提供更好的武器、恢复品或后续拓展标签。

## 9. 系统映射

为保证后续实现可落地，`MVP` 中所有主线步骤都应映射为当前项目已存在的行动与事件类别：

- 坠毁现场、旧飞船遗迹、医疗舱、营地、巢穴：以“前往 + 调查 + 事件通话 / 结果结算”为主
- 丘陵、火山：以“前往 + 采集 / 调查”为主
- 外星村落：以“前往 + 接触对象 + 对应事件 / 交易”为主
- 熔炉、折跃仓：以“前往 + 建设 / 修复”为主
- 雷达：以“调查后解锁新线索 / 新地点”实现，不作为独立复杂子界面

这样可以避免 `MVP` 过早进入复杂的地点内多对象菜单。

## 10. 危险与恢复

- 采集、调查与高风险地点可能触发危险事件。
- 危险事件失败后，队员会获得 `受伤` 标签。
- `受伤` 的效果为移动变慢，并且不能执行高风险行动。
- 恢复手段至少有两种：
  - 使用药品恢复，药品会被消耗
  - 具备 `野外急救` 的队员执行恢复类行动，不消耗药品

两者的恢复效果等价，区别只在于是否消耗资源。

`MVP` 中不做更复杂的长期残废、感染、士气联动。

## 11. 胜利条件

玩家在一次流程中同时满足以下条件，即可完成 `MVP`：

- 已获得 `维修技术`
- 已修复熔炉
- 已制造 `折跃仓修复套件`
- 已获得 `折跃坐标`
- 已获得 `外星粘液燃料`
- 至少存在一名仍可行动、掌握 `维修技术` 的队员
- 所有仍可行动的队员都已回到坠毁区
- 已按顺序完成折跃仓三步：修复仓体结构、注入燃料、输入坐标并确认启动

## 12. Later

以下内容保留世界观与后续扩展价值，但不进入 `MVP` 主线闭环：

- 更复杂的罗塞塔延伸玩法与语言深化系统
- 外星村落的扩展贸易体系（除高温开采设备交易外）
- 驯兽路线
- 女王室交流与蜂群社会知识
- 高处隐秘洞窟、化石与攀爬工具
- 森林菌落、稀有真菌簇与更多制造/贸易配方
- 雷达升级后的更细粒度扫描玩法
- 多星球折跃与后续跨星球内容

## 13. 验收标准

- 玩家在开场后能明确知道主目标是“回家”。
- 玩家总能通过保底路线完成返航，不依赖随机掉落或高属性。
- 每个主线必需道具都存在稳定来源和明确用途。
- 任何一条主线前置都不会被可错过的内容锁死。
- 关键知识标签可以补学，单一队员失效不会直接导致主线失败。
- 医疗舱、村落、营地、巢穴之间的线索链是稳定且可理解的。
- 雷达的主线线索稳定给出，额外发现才允许随机。
- 受伤会产生明确代价，但恢复手段始终存在。
- 高属性路线只提供捷径，不成为唯一解。
