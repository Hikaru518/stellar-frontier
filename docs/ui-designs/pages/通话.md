![Amy 通话草图](../assets/通讯录-通话2.png)

## 页面目标

通话页面用于承载角色事件、现场信息和玩家决策。玩家在这里根据队员的求助内容选择行动，选择结果会影响角色状态、资源状态或[地图](地图.md)上的地块状态。

## User Journey

1. 玩家在[通讯台](通讯台.md)点击“通话”主动联系队员，或在队员来电时点击“接通”。
2. 通话页面显示角色头像、姓名身份、现场图片和台词。
3. 玩家阅读事件描述，判断风险和目标。
4. 玩家选择一个回复或行动按钮。
5. 如果选择需要地点信息，页面可以打开[地图](地图.md)查看坐标、资源、建筑和手下状态。
6. 玩家从地图返回通话页后，在通话页确认行动，系统更新角色状态并给出结果反馈。
7. 玩家结束通话，返回[通讯台](通讯台.md)。

## 页面模块

| 模块 | 显示内容 | 交互职责 |
| --- | --- | --- |
| 通话图片 | 队员现场画面或事件插图。 | 帮助玩家理解危险和环境。 |
| 角色信息 | 头像、姓名、身份。 | 明确是谁在求助。 |
| 台词气泡 | 当前角色台词或事件描述。 | 提供决策信息。 |
| 行动按钮 | 玩家可选择的回复或指令。 | 触发分支、发起指令或结算结果。 |
| 地图二级菜单 | 小型地图入口。 | 从通话中打开地图查看坐标信息。 |
| 通讯录浮层 | 队员列表、来电状态和队员状态。 | 在通话中临时查看其他队员，不中断当前通话。 |

## 通话类型

### 普通状态通话

以玩家主动联系 Garry 的矿床通话为例：

- 角色告知自己正在矿床采矿。
- 玩家可以请求前往其他地点、执行建设、停止工作原地待命或开展调查。
- 如果玩家选择“请求前往”，可以打开地图二级菜单查看目标地块信息；具体前往指令仍在通话页确认。

### 紧急事件通话

以 Amy 主动来电的森林遇熊事件为例：

- 角色发出求救：“头儿，我遇到熊了我草”。
- 玩家需要在有限时间或有限信息下做选择。
- 选择可以体现不同风险：
  - “快跑”：可能导致资源产出中断或丢失当前采集机会。
  - “跟他爆了”：可能击退危险，也可能导致角色受伤或死亡。
  - “先稳住！我考虑一下”：暂缓决策，可能争取信息，也可能让危险升级。

## 可操作项

### 选择回复

- 点击任一回复按钮后进入结果结算。
- 同一轮通话只能选择一次关键决策。
- 选择后按钮应禁用，避免重复结算。
- 不同的事件可以提供选择的选项不同。

### 打开地图二级菜单

- 在需要地点的行动中出现，例如“请求前往”。
- 点击后显示地图预览或进入[地图](地图.md)查看坐标信息。
- 地图页面只展示信息，不提供对手下的指令按钮。
- 玩家查看地块后返回通话页面确认行动。

### 打开通讯录浮层

- 通话中可以打开通讯录浮层，查看其他队员状态和未接通讯。
- 通讯录浮层不替代当前通话的决策，不应触发新的通话结算。
- 玩家关闭通讯录浮层后回到当前通话上下文。

### 执行行动

- “安装/建设”用于在当前地块建立设施。
- “开展调查”用于发现隐藏事件、资源或危险。
- “停止工作原地待命”用于让角色从采集或建设状态切换为空闲/待命。
- 行动成功后应同步更新通讯台队员状态和地图地块状态。

### 结束通话

- 通话结算后显示“结束通话”。
- 点击后返回[通讯台](通讯台.md)。
- 如果事件未决策，结束前需要提示“尚未下达指令”。

## 状态与规则

- 通话页面应持有当前事件上下文：角色、地点、事件类型、可选行动、是否已结算。
- 紧急事件中，选择应产生清晰后果，不建议提供无成本最优解。
- 普通行动中，按钮可根据角色状态、资源和地块条件启用或禁用。
- 如果选择会导致死亡或重大损失，按钮文案可以直接暗示风险，例如“跟他爆了（可能会死）”。
- 地图二级菜单只负责辅助查看地点；对手下的指令仍在通话页面确认。具体地块信息和地图规则见[地图](地图.md)。
- 通话中打开通讯录时，当前通话应保持挂起状态，关闭通讯录后继续处理原事件。

## 示例事件

### Garry：矿床采矿

这是玩家点击“通话”主动联系队员的普通状态通话。

起始状态：

- 角色：Garry，退休老大爷。
- 地点：矿床。
- 行动：采矿中。
- 台词：“头儿，我正在矿床采矿，有什么事吗？”

可选行动：

- 请求前往：打开地图查看目标地块信息，返回通话页后确认前往。
- 执行动作：安装/建设当前可用设施。
- 停止工作原地待命：停止采矿，保留在当前地块。
- 开展调查：调查矿床或附近异常。

### Amy：森林遇熊

这是队员主动联系玩家后，玩家点击“接通”进入的紧急事件通话。

起始状态：

- 角色：Amy，千金大小姐。
- 地点：森林。
- 危险：遭遇野生动物。
- 台词：“头儿，我遇到熊了我草。”

可选行动：

- 快跑：中断资源产出，角色尝试脱离危险。
- 跟他爆了：高风险战斗，可能受伤或死亡。
- 先稳住！我考虑一下：暂缓，可能追加新情报或让危险倒计时推进。
