# 控制中心

![控制中心草图](../assets/控制中心.png)

## 页面目标

控制中心是玩家的基地总入口。玩家在这里观察外部环境、查看基地资源和状态、使用生活设施，并进入通讯台、地图查看系统或后续扩展模块。

## User Journey

1. 玩家进入游戏后默认来到控制中心。
2. 玩家查看当前可交互设施，判断是否有新的通讯或状态变化。
3. 玩家点击通讯台，进入[通讯台](通讯台.md)处理队员事件。
4. 玩家也可以点击窗户、咖啡机、音频终端、冰箱等设施，获得轻量反馈。主要是为了获得沉浸感。
5. 玩家完成操作后回到控制中心，等待下一轮事件或进入其他系统。

## 页面模块

| 模块 | 显示内容 | 设计目的 |
| --- | --- | --- |
| 窗户 | 外部环境入口。 | 图片展示。玩家可以感知星球天气、危险、剧情氛围。 |
| 通讯台 | 通讯系统入口。 | 进入角色列表、来电、通话事件。 |
| 卫星雷达 | 地图入口。 | 查看地图。 |
| 中控台 | 资源、状态、统计。 | 查看当前库存和生产情况。 |
| 咖啡机 | 生活设施。 | 增加沉浸感，触发短文本反馈。 |
| 音频终端 | 背景音乐设施。 | 更换背景音乐或调节氛围。 |
| 冰箱 | 食物设施。 | 吃东西，增加沉浸感，与数值无关，触发短文本反馈。|
| 研究台 | 科技树入口。 | 解锁后续设施、仪器或行动能力。 |
| 星际贸易 | 资源交换入口。 | 用矿物换能源或仪器。 |
| 星际之门 | 总部请求或抽卡入口。 | 获得人员。 |

## 可操作项

### 窗户

- 点击后打开一张外部观察图片或弹窗。
- 弹窗显示当前环境描述，例如沙尘、平原、夜色、远处异常光点。
- 玩家点击关闭后回到控制中心。
- MVP 中只需要展示一张静态图片和一段描述。

### 通讯台

- 点击后进入[通讯台](通讯台.md)独立页面。

### 中控台

- 点击后打开二级窗口，显示资源、基地状态和统计。

### 咖啡机

- 点击后打开二级窗口。
- 玩家可以选择“喝咖啡”。
- 推进一段轻量文本。和数值无关。增加沉浸感。
- 没有资源控制。

### 音频终端

- 点击后打开二级窗口。
- 玩家可以选择背景音乐或切换播放状态。
- 切换后立即关闭窗口或保留窗口让玩家继续选择。

### 冰箱

- 点击后打开二级窗口。
- 玩家可以选择“吃东西”。和数值无关。增加沉浸感。
- 没有资源控制。例如冰箱里有啤酒，那么啤酒会无限量供应。

### 卫星雷达

- 点击后进入地图页面。
- 这是地图的全局查看入口。

### 研究台

- 点击后进入科技树或研究窗口。
- 用于解锁采矿效率、通讯距离、仪器制造等能力。

### 星际贸易

- 点击后进入资源交换窗口。
- 用于将矿物换成食物、能源、仪器或建筑材料。

### 星际之门

- 点击后进入总部请求窗口。
- 玩家可以向总部请求支援或触发抽卡式补给。

## 状态与规则

- 控制中心中的设施默认可点击；未实现功能应给出明确提示，而不是无反馈。
- 重要事件优先通过通讯台提示引导玩家，不应让玩家在多个入口里盲找。
- 二级窗口需要有关闭或返回操作。
- 如果玩家在通话事件进行中返回控制中心，通讯台应保留“通话中”或“待决策”状态。

## 关联页面

- 点击通讯台进入[通讯台](通讯台.md)。
- 点击卫星雷达进入[地图](地图.md)。
- 通讯事件中可能进入[通话](通话.md)。
- 通话行动也可能打开[地图](地图.md)，用于选择行动目标。
- 控制中心自身负责承接各系统返回后的状态变化。
