# 世界设定

本文件用于整理《Stellar Frontier》的世界观设定。

## 核心主题

- 视界线：指挥官永远只能看到部分真相。
- 回家：每一次前进都伴随着代价与取舍。

## 一句话设定

- 一支满载战利品与未知神器的太空冒险队，在返航途中坠毁于陌生星系；你作为后方指挥官，只能依靠不完整情报，一步步带他们穿越多颗星球，拼出回家的坐标航路。

## 基础设定

- 冒险队携带未知神器返航途中遭遇事故，坠落未知星球。
- 玩家扮演基地指挥官，通过通讯与调度支援队员探索。
- 通过逐步收集坐标线索，解锁下一目的地，最终寻找回家路径。

## 世界规则（叙事层）

- 信息不完整是常态：地图、日志、队员口述只能给出部分事实。
- 真相分层揭示：每次任务只解锁一个关键碎片，无法一次看全局。
- 坐标是回家唯一硬通货：资源可替代，坐标不可替代。
- 未知神器是风险放大器：它会放大冲突，但真实作用长期不明。
- 人物成长会影响坐标质量：队员成长越健康，越容易产出高准确度坐标。
- 坐标误差会累积路径代价：误差越大，越可能绕行更多星球。

## 当前局势

- 飞船主引擎损毁，长距跃迁不可用。
- 现有基地只支持短程侦察与中继通讯。
- 队伍必须在资源压力下完成三件事：
  - 生存（维持队员与基地状态）
  - 探索（获取地形与异常情报）
  - 定位（收集下一星球坐标）

## 导航规则（v1）

- 每颗星球任务结束后产出一段候选坐标。
- 坐标有三档质量：高准确 / 可用偏差 / 严重失真。
- 坐标质量由两部分共同决定：
  - 当前执行者能力（属性、专长、状态）
  - 成长状态（信任、创伤、协作稳定度）
- 高准确坐标可缩短归航路径；低准确坐标会触发绕行或错误落点。

## 未知神器（v1）

- 来源：冒险队在上一段旅途中获得，来源记录不完整。
- 外观：存在稳定实体，但其内部结构无法被常规扫描完整解析。
- 已知现象：在高压力事件附近会引发异常信号与环境波动。
- 叙事职责：推动主线，不在前期给出唯一解释。

## 叙事边界

- 主轴是“回家”，不是征服或殖民。
- 决策张力来自“有限视野下的判断”，不是全知全能优化。
- 保持科幻生存基调，允许冷幽默，但不削弱危机感。
