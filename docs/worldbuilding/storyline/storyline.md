# 故事线

## 编写规则

- 故事线不与角色强绑定，剧情故事线和人物成长故事线解耦；文本避免出现具体队员角色姓名，仅保留阵营、地点与系统实体名称。
- 描述需保持故事感，按“情境 -> 冲突 -> 选择 -> 后果”递进，避免只列设定点。
- 关键术语统一标注：机制词使用 `['词汇']`，道具/系统/资源/地点锚点使用 `[词汇]`。

## 冰与火之星

### 第一阶段：坠落与立足

#### 开场局势

飞船在穿越大气层时被撕裂成两段，主舰体拖着火尾撞进冰与火的断层边缘，最终滑停在被当地称为 ['灰烬霜带'] 的交界地表。这里不是中间地带，而是两种极端气候互相撕扯的伤口。

舷窗左侧是会没过脚踝的白色风暴，右侧是把岩石烧到发红的火山雨。环境按固定节律切换 ['霜爆-热浪']：前一分钟结霜，后一分钟灼热。飞船 [发动机核心] 严重受损，[备用电源] 仅能维持短时生命支持，任何外出都在和温差抢命。

[通讯阵列] 充满噪声，[地图] 只剩 ['过期轮廓']，没有 [可靠坐标]，没有友军信号，也没有第二次坠毁机会。队伍此刻要回答的第一个问题不是“去哪里”，而是“谁能活着走出第一小时”。

#### 情境

- 坠毁后的第一小时里，玩家只能依靠 [通讯阵列] 的断续频道指挥队员回收残骸，拼出临时保温层、散热片和简易中继。
- 在 ['灰烬霜带']，每一次外出都被 ['霜爆-热浪'] 追着跑，探索不是冒险浪漫，而是用物资换生存时间。

#### 冲突

- 队伍必须尽快接触当地村庄拿到区域生存装备，否则无法进入冰区或火区深处。
- 但当前情报互相矛盾：
  - 冰侧出现不自然反光与耐寒兽拖痕；
  - 火侧出现等距火点与矿渣辙痕；
  - [地图] 仅有 ['过期轮廓']，无法确认哪条路更安全。

#### 选择

- 玩家做第一轮派遣决策：
  - 全员去冰村（稳健集中）
  - 全员去火村（激进集中）
  - 分两队并行（信息最大化）
- 无论哪种路线，都必须抓住短暂温差回落窗口穿越交界带；错过窗口会触发额外消耗或状态惩罚。

#### 后果

- 全员去冰村：生存稳定性更高，但火区情报明显滞后，主线推进偏慢。
- 全员去火村：高能材料获取更快，但事故与损耗风险显著上升。
- 分队并行：冰火两侧信息同步增长，利于后续拼图，但通讯调度复杂且容错最低。
- 抵达任一村庄后，村民不会立刻信任外来者；玩家需用资源、工具或劳务换取落脚资格，才可解锁基础装备与地方传闻。

#### 阶段收束（第一阶段产出）

- 至少一条稳定补给线（冰村或火村）。
- 一套可持续的双环境基础装备方案。
- 第一段不完整坐标线索（需后续阶段校准）。
- 对 [麦克斯韦妖精] 的初步认知：强大、难以正面对抗、可能在守卫 [门域]。

#### 阶段目标（复盘）

- 在冰火交界的极端环境中建立最低生存能力。
- 与至少一个村庄建立联系，获得区域生存装备与本地情报。
- 拿到第一批“可疑但不完整”的坐标信息。

### 第二阶段：双区调查与生态战斗

#### 开场局势

队伍在第一阶段站稳脚跟后，开始向冰区与火区深处推进。很快，回传信息出现明显矛盾：冰原巡线报告“被搬运后留下的低温空带”，火山巡线报告“重复高温脉冲与矿层塌陷”。村庄仍坚持 [麦克斯韦妖精] 是灾变源，但现场痕迹更像有规律运输，而非无差别破坏。

与此同时，队员遭遇第一批区域性怪物。它们不只是强弱差异，而是被环境重新塑形：冰区生物偏耐寒与群猎，火区生物偏高温与侵袭。玩家逐渐意识到，想理解这个星球，必须先理解它被气候撕裂后的生物系统。

#### 情境

- 调查队需要在冰区与火区同步采集线索，拼出可用的 [坐标碎片]。
- 怪物战斗成为主线推进的一部分：关键样本与装备材料只能通过狩猎或防卫战获得。
- 村庄开始发布狩猎委托，承诺用工艺、情报和通行权限交换战斗成果。

#### 冲突

- 资源与时间不足以支持“无差别探索”，玩家必须决定优先调查方向与狩猎目标。
- 若缺少对应区域材料，队伍会在冰区/火区持续吃环境惩罚。
- 若战斗投入过高，主线调查节奏会被拖慢；若战斗投入过低，装备与样本不足又会卡住主线。

#### 选择

- 区域推进策略：
  - 偏冰区深挖：先保障耐寒体系与冷端样本。
  - 偏火区深挖：先保障耐热体系与高能样本。
  - 双区同步校验：并行采样，提升信息完整度但增加调度压力。

- 战斗与委托策略：
  - 冰区狩猎线：
    - [霜脊狼] -> 掉落 [霜狼皮]（耐寒内衬）
    - [裂冰甲虫] -> 掉落 [冰壳板]（低温护甲部件）
  - 火区狩猎线：
    - [熔鳞蜥] -> 掉落 [熔鳞]（耐热层）
    - [硫烟飞螳] -> 掉落 [硫囊]（过滤与驱散组件）
  - 交界稀有线：
    - [相变捕食者] -> 掉落 [相变膜]（双环境关键材料）

- 村庄支线选择：
  - 承接冰村委托（换取冰区工艺与捷径情报）
  - 承接火村委托（换取火区通行许可与高能材料）
  - 同时承接两边（收益高，但时间窗口压力更大）

#### 后果

- 偏冰区深挖：环境生存更稳，但火端参数不足，主线推进偏慢。
- 偏火区深挖：工程收益更快，但事故率与装备损耗明显上升。
- 双区同步校验：最容易拼出高质量 [坐标碎片]，但任何调度失误都可能引发连锁崩盘。
- 完成村庄委托会提升当地信任，并解锁下一阶段所需的 [门域] 外围情报入口。

#### 阶段收束（第二阶段产出）

- 获取冰端与火端的关键样本。
- 解锁至少一条村庄高信任回报（工艺/通行/情报）。
- 拼出可校准的 [坐标碎片]。
- 明确第三阶段目标：进入门域外围，验证妖精守卫逻辑并补全离场坐标条件。

#### 阶段目标（复盘）

- 通过战斗与调查同时建立“生态理解 + 装备升级 + 坐标拼图”。
- 让玩家直观看到生物多样性与环境变迁的因果关系。
- 以村庄支线回报反哺主线推进，形成调查与战斗的闭环。

### 第三阶段：门域接触与分歧

#### 开场局势

第二阶段结束后，队伍已经拿到可校准的 [坐标碎片]，却仍缺少离场所需的关键参数。所有线索都指向 [门域]：那里是 [麦克斯韦妖精] 的活动核心，也是唯一可能补全离场条件的区域。

但队伍不能再靠“单一路径”推进。村庄、队员与后勤状态都在逼迫玩家站队：是潜入、沟通，还是驱逐。第三阶段由三条支线组成，每条支线对应一种策略辅助。

在对门域残留记录的破译中，队伍首次接触到妖精文明的核心信念：它们信奉 ['智慧之神'] 与 ['多维宇宙之神']。传说这两位神赐予了 [门] 的科技，包括用于逆熵实验的核心门与跨时空的 [传送网络]。这让“沟通路线”不再只是外交尝试，而是一次接近神话级技术体系的风险接触。

#### 情境

- 主线目标：补全离场参数并进入门域外围。
- 推进方式：完成三条策略支线中的任意两条，解锁终局入口。
- 规则：不同支线组合将改变终局默认策略倾向与风险结构。

#### 冲突

- 时间窗口有限，无法三线完美推进。
- 资源不足以同时支持“高强度战斗 + 复杂研究 + 多线侦察”。
- 每条支线都会放大某种能力，也会放弃另一种冗余安全。

#### 选择

1. 支线 A：盲区测绘（辅助潜入策略）
   - 目标：定位 [门域] 巡逻网的短时盲区，建立潜入窗口。
   - 关键行动：拼接冰村风向记录、火村矿道旧图与外围巡逻时间表。
   - 成功收益：解锁“低警戒潜入”选项，潜入失败惩罚降低。
   - 失败后果：触发追击事件，后续潜入警戒值上升。

2. 支线 B：七拍语法（辅助沟通策略，复杂线）
   - 目标：建立与 [麦克斯韦妖精] 的最低可用交流协议。
   - 四步流程：
     - 采样：在三处点位记录 ['七拍回声']，并完成昼夜双时段采样。
     - 对照：结合遗迹碑文与村庄口述，完成“节律-行为”映射并剔除误译。
     - 构建：狩猎稀有目标获取 [共振腺体]，制作 [共振发声器]。
     - 实测：在门域边缘执行三轮“信号-回应”测试（让路/停火/交换），并完成一次 ['神名誓约'] 语句校验。
   - 成功收益：解锁“协议通行”选项，门域敌意显著下降，并获得关于 ['智慧之神'] / ['多维宇宙之神'] 与 [传送网络] 的片段真相。
   - 失败后果：进入“误译状态”，短期关闭沟通选项并提升警戒。

3. 支线 C：边境清剿（辅助驱逐策略）
   - 目标：削弱妖精外围控制圈，争取强行操作窗口。
   - 关键行动：接受村庄联合作战委托，清除外围节点并守住回收点。
   - 成功收益：解锁“强制介入门域”选项，短时夺取控制权。
   - 失败后果：队伍状态恶化、村庄信任下降、生态波动升级。

#### 后果

- 完成任意两条支线后进入终局任务。
- 组合结果会影响终局策略倾向：
  - A + B：潜入/沟通优势，整体风险更可控。
  - A + C：潜入/驱逐混合，执行力强但波动高。
  - B + C：可谈可打，灵活但资源压力最大。
- 若仅完成一条支线，主线可继续但会带着明显短板进入高风险状态。
- 走沟通优势（A+B 或 B+C）时，可触发“神话-技术双解释”分支：
  - 村民视角：妖精是在执行神谕。
  - 探险队视角：神谕可能是对高维技术规则的人格化叙述。

#### 阶段收束（第三阶段产出）

- 至少两条策略支线的可用成果。
- 一套明确的门域介入路径（潜入 / 沟通 / 驱逐或混合）。
- 离场参数补全过程所需的前置条件清单。

#### 阶段目标（复盘）

- 让策略选择从“偏好”变成“有代价的结构化取舍”。
- 用支线把三种策略具象化，并为终局路径提供可解释前置。

### 第四阶段：离场窗口与终局解释

#### 开场局势

第三阶段结束后，队伍终于具备介入 [门域] 的条件，但窗口极短。['霜爆-热浪'] 周期进一步压缩，门域稳定时间开始跳变，飞船 [发动机核心] 只能承受一次高负荷启动。玩家必须在一次执行中完成三件事：补全离场参数、完成引擎修复、锁定 [下一星球坐标]。

与此同时，关于妖精与 [门] 的解释出现分裂：
- 神谕解释：妖精在执行 ['智慧之神'] / ['多维宇宙之神'] 的意志。
- 技术解释：所谓神谕是高维技术规则的人格化叙述。

#### 情境

- 终局不再提供新线索，而是结算前三阶段的全部选择。
- 玩家需决定采用哪条终局执行策略（潜入 / 沟通 / 驱逐）。
- 每条策略都能让飞船离场，但带走的“真相深度”和“代价结构”不同。

#### 冲突

- 时间只够执行一套主方案，无法兼顾所有价值。
- 玩家必须在“回家成功率 / 队伍完整度 / 星球后续稳定”之间排序。
- 任何方案都不是零代价，区别只在代价落到谁身上、留到何时爆发。

#### 选择

1. 潜入主导终局（信息优先）
   - 执行：利用盲区窗口潜入门域外围，提取 [逆卡诺校准式]、[门芯相位钥] 与关键时序参数。
   - 特点：获取真相速度快，但只能拿到“可复现方案”，无法直接带走门域核心。
   - 代价：暴露后警戒飙升，离场窗口可能被强行压缩。

2. 沟通主导终局（协议优先）
   - 执行：通过“七拍语法 + ['神名誓约']”与妖精达成临时通行协议，获得授权参数与稳定供能窗口。
   - 特点：最稳路径，可降低门域敌意并减少生态冲击。
   - 代价：流程最长，对前置支线完整度要求最高。

3. 驱逐主导终局（控制权优先）
   - 执行：联合村庄发动强制清剿，夺取门域短时控制权并强行介入修复流程。
   - 特点：短期效率高，可直接拿到足够修复资源。
   - 代价：队伍损耗和生态波动显著上升，且真相层最薄。

#### 后果

- 终局会同时给出“离场结果”与“解释立场”。

- 结局 A：速返（潜入倾向）
  - 离场：成功，航程最短。
  - 解释：偏技术解释，神谕被视为编码叙事。
  - 代价：门域关系紧张，星球后续不稳定。

- 结局 B：协返（沟通倾向）
  - 离场：成功，航程中等。
  - 解释：神谕与技术并存，保持“可解释但不被解释完”。
  - 代价：时间成本高，但队伍与星球状态最稳。

- 结局 C：血返（驱逐倾向）
  - 离场：勉强成功。
  - 解释：真相未解，叙事停在“胜利但空白”。
  - 代价：高伤亡/高创伤，后续章节负担最大。

- 结局 D：滞留修正（终局失败）
  - 离场：失败，但飞船未彻底毁坏。
  - 解释：无定论，进入补救篇章。
  - 代价：资源重置压力与窗口再争夺。

#### 阶段收束（第四阶段产出）

- [下一星球坐标] 与对应置信标签。
- 当前章节解释立场标签（神谕倾向 / 技术倾向 / 混合倾向）。
- 星球遗留状态标签（友好 / 紧张 / 破碎），用于后续章节继承。

#### 阶段目标（复盘）

- 把“回家”从单一胜负改为“可达成但带后果”的终局结构。
- 通过神谕解释与技术解释并存，保留世界观神秘感与讨论空间。

### 支线库

以下支线可按阶段插入，默认与主线解耦，但会通过资源、情报与阵营关系反哺主线。

#### 支线 01：失温婚约（冰村）

- 情境：两名村民被困在冰线两端，委托队伍护送 ['热石誓物'] 完成会合。
- 冲突：护送路线必须穿过 ['霜爆-热浪'] 高发带，时间窗口很短。
- 选择：走长线低风险路径，或走短线高风险路径。
- 后果：完成后解锁 [冰区捷径坐标] 与补给折扣；失败会降低冰村信任。

#### 支线 02：火井税单（火村）

- 情境：火村要求清除矿井周边高危生物，才开放锻造炉权限。
- 冲突：主线也需要同批材料，战斗收益与主线材料发生竞争。
- 选择：优先交付村庄换权限，或保留材料优先推进主线。
- 后果：解锁 [耐热锻造许可]、[熔鳞] 加工能力，或保留即时主线推进速度。

#### 支线 04：灰烬拾荒者（交界带）

- 情境：一支拾荒队在 ['灰烬霜带'] 失联，仅剩断续求救信号。
- 冲突：救援会占用主线侦察窗口，放弃则可能失去关键线索。
- 选择：全力救援、有限救援、放弃救援。
- 后果：救援成功可获得 [相变膜] 线索与旧航图碎片；放弃会带来长期舆情负面。

#### 支线 05：孢子禁区（门域外围）

- 情境：村庄请求封锁一处 [孢子禁区]，避免扩散到居住区。

#### 支线 09：双村停火（社会线）

- 情境：冰村与火村因资源分配进入对抗，双方都要求探险队站队。
- 冲突：站队可换即时高回报，但会破坏长期协作网络。
- 选择：促成停火、偏向冰村、偏向火村。
- 后果：停火成功可解锁 [双村联合补给网]；单边站队会获得短期资源爆发与长期关系代价。

#### 支线 10：最后一班矿车（战斗撤离）

- 情境：火区矿道塌陷，最后一班矿车被困在高温走廊。
- 冲突：需边战斗边护送撤离，且 [备用电源] 与冷却资源不足。
- 选择：先清场后撤离，或边打边撤。
- 后果：成功可获得高能矿样本与火村作业支持；失败会造成队伍状态恶化与火村信任下降。
